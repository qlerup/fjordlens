# FjordLens Icons

Drop your exported PNGs/SVGs into this folder with the following filenames so browsers can pick them up automatically:

Required (recommended):
- favicon-16x16.png (16×16)
- favicon-32x32.png (32×32)
- apple-touch-icon.png (180×180)
- android-chrome-192x192.png (192×192)
- android-chrome-512x512.png (512×512)

Optional:
- android-chrome-512x512-maskable.png (512×512, maskable)
- safari-pinned-tab.svg (monochrome SVG)
- mstile-150x150.png (Windows tile)

Already included as a functional fallback:
- fjordlens.svg — a simple SVG used as default favicon.

The head links are already wired in templates/index.html.
